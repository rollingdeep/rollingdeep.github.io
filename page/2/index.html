问题： Windows 下文件夹占用解决办法
解决办法：

  1. 按下 win 键
  2. 搜索”资源监视器”， 打开
  3. 选择 CPU 栏目
  4. 在”关联的句柄” 搜索框输入 文件夹的名称
  5. 鼠标右键，结束进程。
